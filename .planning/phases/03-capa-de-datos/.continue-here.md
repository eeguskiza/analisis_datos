## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Bind-mount missing para artefactos generados en container | El baseline PDF de 03-02 se grabó dentro del container y se perdió al `docker compose up -d --build web` porque `tests/` no estaba mounted. Hubo que repetir trabajo del operador. | advisory | `docker-compose.yml` ya tiene `./tests:/app/tests` mounted (commit `c0cde5c`). Para nuevos artefactos generados en container que deban persistir entre rebuilds, verificar bind-mount al añadir scripts. |
| Plan asume contrato incorrecto de función legacy | Los scripts PDF (`scripts/gen_pdf_reference.py` + `scripts/pdf_regression_check.py`) asumían que `run_pipeline()` aceptaba `output_dir` cuando en realidad es un generator que escribe a `settings.informes_dir`. Causó error en Task 2 del 03-02. | advisory | Antes de generar plans con código nuevo que llame a funciones existentes, leer la signature real de la función (no asumir). Pattern-mapper agent ya hace esto pero scripts en la raíz a veces se escapan. |
